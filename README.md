# NiaServer-Extra

NIA服务器额外扩展玩法 **“海洋限定版 V5”** 相关实现代码

## 写在前面

本项目为[NiaServer-Core](https://github.com/Nia-Server/NiaServer-Core)子项目，项目主体文件内容为Minecraft基岩版的addons附加包。

**启用存档中所有BETA选项以及教育版模式**
